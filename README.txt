App Description:
Uses the Pokemon API to search for pokemon which are added to the user's pokemon collection via a MongoDB database.
The user is able to view and modify their pokemon by performing evolutions, add new ones and remove old ones.

Expected Updates:
Chance a pokemon is shiny when searched in the PokeAPI.
Branched evolution, i.e. choose if eevee evolves into jolteon, vaporeon, etc.
Abilty to give a pokemon a nickname.
Bug fixes and security/UI updates.
